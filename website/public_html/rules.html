How to Play

If you haven't already, head over to the API page to learn how to connect your robot with Scavenger Hunt.

This site was designed with the intention to track the progress of different universities in their robot development by creating a standardized set of tasks designed to be done by robots. These tasks can be put into "hunts" that will test the robot in how well it can not only implement each task, but complete the hunt in the most efficient way possible.

We will be publishing a daily hunt entitled "Dijkstra Hunt" that test the robot on relatively simple tasks. This will help you work on creating ways to make the robot complete tasks efficiently. For example by working on multiple tasks at once, or possibly developing a machine learning algorithm so the robot can finish a task more quickly.

We will also be publishing a weekly hunt entitled "Turing Hunt" that will test the robot on relatively difficult tasks. This will foster an exapnsion in research with tasks that may have not been implemented yet. The goal of this is to hopefully allow labs to develop new programs while knowing that there are other labs trying to do the same.

Both of these hunts will have a number associated with it ("Dijkstra Hunt # or Turing Hunt #") which you must use to access the current hunt. We will also be publishing a leaderboard for these hunts that display how many tasks were attempted, the success rate, and score obtained. These are updated continuously so there may be ongoing hunts alongside completed hunts. The goal is to achieve the highest score, and by doing so have the best success rate. One can also look at the points per task, but be careful as the ones with a higher point value will also be much more difficult to implement and accomplsih consistently.

Good luck to everyone!
